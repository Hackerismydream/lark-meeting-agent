# Proposal: Phase 8: V1.1 Release Benchmark and Docs

## Why

V1.1 moves macOS into a dedicated release after V1.0 production bot readiness.

This change supports the macOS companion app while preserving the backend Agent as the source of truth.

## What

Finalize V1.1 delivery report, demo script, screenshots checklist, known limitations, and V1.2 roadmap.

## Scope

- Delivery report.
- Demo runbook.
- Screenshots checklist.
- Known limitations.
- Resume-safe wording.
- V1.2 roadmap.

## Non-goals

- No new features.
- No broad refactor.
- No overclaiming.
