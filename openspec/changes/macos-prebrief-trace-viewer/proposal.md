# Proposal: Phase 5: PreBrief, Meeting Timeline, and Run Trace Viewer

## Why

V1.1 moves macOS into a dedicated release after V1.0 production bot readiness.

This change supports the macOS companion app while preserving the backend Agent as the source of truth.

## What

Add views for today's meetings, pre-briefs, live session state, run traces, audit events, tool calls, and write results.

## Scope

- Show meeting list.
- Show pre-brief with sources.
- Show run detail and status.
- Show trace timeline.
- Show audit/write results.
- Show error categories.

## Non-goals

- No complex dashboard.
- No production observability dashboard inside app.
- No hidden Lark writes.
