# Proposal: Codex Computer Use Demo Evidence

## Summary

Create a demo evidence pass that uses UI screenshots to show local artifacts and sandbox-ready outputs from the Lark Meeting Agent workflow.

## Scope

- Product demo evidence only.
- Screenshots of local artifacts and sandbox/dry-run previews.
- Blockers for unavailable or privacy-sensitive real Feishu UI steps.
- `docs/DEMO_EVIDENCE_REPORT.md` summarizing screenshots, data sources, dry-run boundaries, and blockers.

## Non-goals

- No new Agent functionality.
- No benchmark replacement.
- No real writes.
- No secret entry.
- No fabricated screenshots.
