# V1 Known Limitations

- No custom ASR or audio ingestion is implemented.
- No macOS companion app exists in V1.
- No production Feishu channel deployment is verified.
- No real OpenAPI tenant smoke is verified.
- No real live meeting smoke success is claimed without a meeting number.
- Optional real LLM benchmark is not included in the V1 release report.
- Vector search and richer RAG remain future work.
- Admin deletion CLI is documented but not implemented.
- OpenTelemetry/Phoenix export is documented as optional but not implemented.
